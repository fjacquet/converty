# Milestone v2.0: Network Tools & User Experience

**Status:** ✅ SHIPPED 2026-01-22  
**Phases:** 9-16  
**Total Plans:** 22

## Overview

v2.0 expanded the Converty calculator platform with comprehensive network tools featuring a visual subnet calculator, added global search functionality for discovering 200+ calculators, and achieved 100% translation coverage across all 4 locales (English, French, German, Italian). This milestone eliminated all hardcoded English strings from calculator components and introduced new categories of tools for IT professionals and network engineers.

## Milestone Summary

**Key Decisions:**

- Used ipaddr.js for IP address manipulation (battle-tested, 55M+ weekly downloads)
- Network as separate category from data (allows future growth)
- Featured subnet calculator on homepage (first network tool showcase)
- BigInt for IPv6 host calculations (exceeds Number.MAX_SAFE_INTEGER)
- RFC 3021 compliance for /31 point-to-point subnets
- Fuse.js for client-side fuzzy search (~5KB gzipped)
- cmdk for Command palette UI
- Pre-built search indexes per locale (build-time JSON generation)
- Translation keys returned from converters (not display strings)
- Wave-based execution for Phase 16 parallel translation work

- ✅ Network category created with proper metadata and translations
