### Task 2: Externalize DateTime - Time Units with Pluralization

**Files:**

- `src/lib/converters/datetime/day-counter.ts`
- `src/lib/converters/datetime/duration-converter.ts`
- `src/lib/converters/datetime/time-duration.ts`

Change to return count + unit key, let UI handle pluralization:

```typescript
// Before
return "3 weeks and 2 days"

// After
return { weeks: 3, days: 2 }  // UI formats with ICU pluralization
```

**Translation keys (ICU format):**

```json
{
  "calculator": {
    "datetime": {
      "timeUnits": {
        "year": "{count, plural, one {# year} other {# years}}",
        "month": "{count, plural, one {# month} other {# months}}",
        "week": "{count, plural, one {# week} other {# weeks}}",
        "day": "{count, plural, one {# day} other {# days}}",
        "hour": "{count, plural, one {# hour} other {# hours}}",
        "minute": "{count, plural, one {# minute} other {# minutes}}",
        "second": "{count, plural, one {# second} other {# seconds}}"
      },
      "and": "and"
    }
  }
}
```
