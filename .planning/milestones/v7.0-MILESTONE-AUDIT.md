## Key Architectural Decisions

- **Test environment:** `node` (not jsdom) for converter tests — ~200ms faster per file
- **Coverage scope:** `src/lib/converters/**/*.ts` only — excludes React layer
- **Zod schema type:** `z.string().refine()` pattern (not `z.coerce.number()`) to match string-typed FormValues
- **URL compression:** Search params `?z=` (not hash) — GitHub Pages compatibility
- **Discriminated union:** Adapter pattern — stores stay backward compatible, unwrap internally
- **i18n schema:** 4-key: `common`, `nav`, `converter`, `calculator` — stable, non-overlapping
- **CI test command:** `npm run test:run` (not `npm test`) — avoids interactive watch mode hanging CI
