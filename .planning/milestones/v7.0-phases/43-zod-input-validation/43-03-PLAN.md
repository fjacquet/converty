  <files>src/lib/schemas/finance/index.ts</files>
  <action>
Create the directory `src/lib/schemas/finance/` and file `src/lib/schemas/finance/index.ts`.

Read each finance calculator component to determine its FormValues/input type, then write the matching Zod schema.

Finance category has TWO patterns:
1. String-based FormValues (most finance calculators): use `z.coerce.number()`
2. Direct number types (compound-interest, retirement store uses numbers directly): use `z.number()`

Check the store's `initialValues` to determine the pattern. If `initialValues: { principal: 10000 }` (number), use `z.number()`. If `initialValues: { principal: "10000" }` (string), use `z.coerce.number()`.

Finance-specific range constraints:
- Interest rates: `.min(0).max(100, "Rate must be between 0 and 100%")`
- Principal/amounts: `.positive("Must be positive")` — no upper bound needed
- Loan terms in years: `.int().min(1).max(50, "Term must be 1–50 years")`
- Loan terms in months: `.int().min(1).max(600, "Term must be 1–600 months")`
- Percentages (down payment, deposit %): `.min(0).max(100)`
- Periods/years for retirement: `.int().min(1).max(80)`
- Frequency enums: `z.enum(["monthly", "quarterly", "annually", "daily", "weekly"])` — match actual enum values used by each calculator

Calculators to cover (read each component to get its exact FormValues type):
annuity-calculator, auto-loan, bond-calculator, break-even, compound-interest, credit-card,
currency (converter), debt-payoff, discount, down-payment, home-equity, inflation, ira-calculator,
loan, mortgage, personal-loan, profit-margin, retirement-401k, retirement, roi,
savings-goal, student-loan, tip

NOTE on compound-interest: Check whether it uses string or number fields — it may use `z.number()` directly (not coerce) since it historically used CompoundInterestInput type directly.

Naming convention: ComponentName + "FormSchema" in PascalCase — e.g., CompoundInterestFormSchema, MortgageFormSchema.

Biome: no `any`, single `import { z } from "zod"` at top.

Finance-specific rules:
