  <files>src/lib/utils/url-params.ts</files>
  <action>
Create src/lib/utils/url-params.ts with three type-safe parsing functions:

1. **parseNumberParam(value: string | null, fallback: number): number**

   - Return fallback if value is null or empty string
   - Parse using Number(value)
   - Return fallback if Number.isNaN(parsed)
   - Otherwise return parsed number
   - Use Number.isNaN() (NOT isNaN() - no type coercion)

2. **parseStringParam(value: string | null, fallback: string): string**

   - Return fallback if value is null or empty string
   - Otherwise return value

3. **parseBooleanParam(value: string | null, fallback: boolean): boolean**
   - Return fallback if value is null or empty string
   - Return true if value === "true" || value === "1"
   - Return false otherwise

Add JSDoc comments for each function with @param and @returns.
Include inline usage examples showing URLSearchParams integration.
Follow existing codebase conventions: double quotes, 2-space indent, semicolons.
</action>
