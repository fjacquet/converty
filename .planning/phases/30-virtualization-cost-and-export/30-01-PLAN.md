  <action>
Create `src/lib/converters/infrastructure/virtualization-cost.ts` with:

1. **Constants for default values:**
   ```typescript
   const DEFAULT_PUE = 1.5; // Typical data center Power Usage Effectiveness
   const HOURS_PER_YEAR = 8760;
   ```

2. **Input interface `VirtualizationCostInput`:**
   - Hardware costs (CAPEX):
     - `serverCost`: number - Total cost of servers ($)
     - `storageCost`: number - Total cost of storage ($)
     - `networkCost`: number - Total cost of network equipment ($)
   - Software costs (annual OPEX):
     - `vmwareLicenseCost`: number - Annual VMware licensing ($)
     - `osLicenseCost`: number - Annual OS licensing ($)
     - `backupSoftwareCost`: number - Annual backup software ($)
   - Operational costs:
     - `powerCostPerKwh`: number - Electricity cost ($/kWh)
     - `totalPowerKw`: number - Total power consumption (kW)
     - `pue`: number - Power Usage Effectiveness (typically 1.3-2.0)
     - `datacenterCostPerRu`: number - Datacenter cost per rack unit per month ($)
     - `totalRackUnits`: number - Total rack units used
     - `laborCostAnnual`: number - Annual labor/admin cost ($)
   - Configuration:
     - `vmCount`: number - Total VMs in environment
     - `termYears`: 1 | 3 | 5 - Analysis term in years

3. **Result interface `VirtualizationCostResult`:**
   - `capex`: number - Total capital expenditure (hardware)
   - `opexAnnual`: number - Total annual operating expenditure
   - `opexTotal`: number - Total OPEX over term
   - `tco`: number - Total Cost of Ownership
   - `costPerVm`: number - TCO per VM over term
   - `costPerVmMonthly`: number - Monthly cost per VM
   - `breakdown`: object with:
     - `hardware`: number - Total hardware (CAPEX)
     - `software`: number - Total software over term
     - `power`: number - Total power cost over term
     - `datacenter`: number - Total datacenter cost over term
     - `labor`: number - Total labor cost over term
   - `percentages`: object with:
     - `hardware`: number - % of TCO from hardware
     - `software`: number - % of TCO from software
     - `power`: number - % of TCO from power
     - `datacenter`: number - % of TCO from datacenter
     - `labor`: number - % of TCO from labor
   - `steps`: string[] - Calculation breakdown

4. **Calculation function `calculateVirtualizationCost`:**
   - Validate all inputs are non-negative
   - Validate PUE is >= 1.0 (physical minimum)
   - Validate vmCount > 0 and termYears is 1, 3, or 5
   - Calculate CAPEX: serverCost + storageCost + networkCost
   - Calculate annual power cost: totalPowerKw * pue * HOURS_PER_YEAR * powerCostPerKwh
   - Calculate annual datacenter cost: datacenterCostPerRu * totalRackUnits * 12
   - Calculate annual software cost: vmwareLicenseCost + osLicenseCost + backupSoftwareCost
   - Calculate annual OPEX: power + datacenter + software + labor
   - Calculate total OPEX: opexAnnual * termYears
   - Calculate TCO: capex + opexTotal
   - Calculate cost per VM: tco / vmCount
   - Calculate monthly cost per VM: (tco / termYears / 12) / vmCount
   - Build breakdown and percentage objects
   - Return null for invalid inputs

Include comprehensive JSDoc comments explaining formulas.
