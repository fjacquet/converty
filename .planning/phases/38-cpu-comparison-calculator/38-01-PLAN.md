**Interfaces to export:**

```typescript
export interface CpuComparisonInput {
  /** Comma-separated CPU IDs for URL sync, e.g. "intel-xeon-platinum-8592plus,amd-epyc-9654" */
  cpuIds: string;
  /** Vendor filter: "all" | "intel" | "amd" | "arm" */
  vendor: string;
  /** Generation filter: "all" | "current" | "previous" */
  generation: string;
}

export interface CpuComparisonRow {
  id: string;
  name: string;
  vendor: CpuVendor;
  family: string;
  generation: CpuGeneration;
  cores: number;
  tdpW: number;
  socketType: string;
  specint2017Base: number;
  specint2017Peak: number;
  /** specint2017Peak / cores, rounded to 2 decimal places */
  perfPerCore: number;
  /** specint2017Peak / tdpW, rounded to 2 decimal places */
  perfPerWatt: number;
  /**
   * Sizing ratio vs the first selected CPU.
   * For CPU[0]: always 1.0
   * For CPU[N]: cpu[0].specint2017Peak / cpuN.specint2017Peak
   * Interpretation: "You need N units of this CPU to match CPU[0]'s performance"
   */
  sizingRatioVsFirst: number;
  /** Optional notes from the database entry */
  notes?: string;
}

export interface CpuComparisonResult {
  rows: CpuComparisonRow[];
  /** The baseline CPU (first selected) */
  baselineCpuId: string;
  /** ISO date string of database data freshness */
  dataAsOf: string;
  /** Whether the data is older than staleDays */
  isStale: boolean;
  /** Human-readable staleness warning, only populated when isStale is true */
  staleWarning?: string;
}
```

**Functions to export:**

1. `getFilteredCpus(vendor: string, generation: string): CpuEntry[]`
   - Reads `cpuDatabase.cpus`
   - Filters: if vendor !== "all", keep only entries where `entry.vendor === vendor`
   - Filters: if generation !== "all", keep only entries where `entry.generation === generation`
   - Returns sorted array: sort by `specint2017Peak` descending (highest first)

2. `calculateCpuComparison(input: CpuComparisonInput): CpuComparisonResult | null`
   - Parse `input.cpuIds` by splitting on comma, trim each, filter empty strings
   - Look up each ID in `cpuDatabase.cpus`; skip IDs not found
   - If resolved CPU array length < 2, return null
   - If resolved CPU array length > 4, use only the first 4 (clamp)
   - For each resolved CPU at index `i`, build a `CpuComparisonRow`:
     - `perfPerCore = Math.round((cpu.specint2017Peak / cpu.cores) * 100) / 100`
     - `perfPerWatt = Math.round((cpu.specint2017Peak / cpu.tdpW) * 100) / 100`
     - `sizingRatioVsFirst`: if i === 0 → 1.0; else → `Math.round((cpus[0].specint2017Peak / cpu.specint2017Peak) * 100) / 100`
   - Compute staleness: `const daysOld = Math.floor((Date.now() - new Date(cpuDatabase.dataAsOf).getTime()) / 86400000)`; `isStale = daysOld > cpuDatabase.staleDays`
   - Return `CpuComparisonResult` with rows, baselineCpuId = cpus[0].id, dataAsOf, isStale, and staleWarning (set to `cpuDatabase.staleWarning` if isStale, else undefined)

Use strict TypeScript — no `any` types. All types from cpu-types.ts or explicitly declared.
