Create subnetting.ts with the divideSubnet() function that divides a network into smaller equal subnets.

**Interface to export:**
```typescript
export interface SubnetDivision {
  parent: SubnetResult;      // Original subnet
  children: SubnetResult[];  // Array of child subnets
  divisions: number;         // Number of divisions (power of 2)
  newCidr: number;          // New CIDR prefix for children
}
```

**Function signature:**
```typescript
export function divideSubnet(
  networkAddress: string,
  cidr: number,
  divisions: number
): SubnetDivision
```

**Algorithm requirements:**
1. Validate divisions is power of 2: `(n & (n - 1)) === 0 && n > 1`
2. Calculate bits needed: `Math.log2(divisions)`
3. Calculate new CIDR: `cidr + bitsNeeded`
4. Validate new CIDR doesn't exceed limits (32 for IPv4, 128 for IPv6)
5. Use existing calculateSubnet() for parent calculation
6. Calculate each child subnet by adding offset to base address:
   - Block size = 2^(maxBits - newCidr) where maxBits is 32 for IPv4, 128 for IPv6
   - Child i address = networkAddress + (blockSize * i)

**Helper functions to implement:**
- `isPowerOfTwo(n: number): boolean` - Bitwise check
- `getBlockSize(ipVersion: 4 | 6, cidr: number): bigint` - Calculate address block size
- `addToAddress(address: string, offset: bigint): string` - Add offset to IP address (handles IPv4 and IPv6)

**Error handling:**
- Throw Error if divisions is not power of 2
- Throw Error if new CIDR exceeds maximum

Use ipaddr.js for IP address parsing and manipulation. Import calculateSubnet from ./subnet-calculator.

Create supernetting.ts with the aggregateNetworks() function that combines contiguous networks into a larger CIDR block.

**Interface to export:**
```typescript
export interface SupernetResult {
  supernet: SubnetResult;           // Aggregated supernet
  originalNetworks: SubnetResult[]; // Original networks
  success: boolean;                 // Whether aggregation succeeded
  error?: string;                   // Error message if failed
}
```

**Function signature:**
```typescript
export function aggregateNetworks(networks: string[]): SupernetResult
```

**Algorithm requirements:**
1. Validate minimum 2 networks
2. Validate count is power of 2
3. Parse all networks (must be valid CIDR notation)
4. Validate all same IP version (IPv4 or IPv6)
5. Validate all same CIDR prefix
6. Sort networks by address (numerically)
7. Calculate new CIDR: `originalCidr - log2(networkCount)`
8. Validate first network is on supernet boundary:
   - Calculate expected supernet network address using ipaddr.js networkAddressFromCIDR
   - Must equal first network's address
9. Validate networks are contiguous:
   - Each network must start exactly where previous ends
   - Use block size calculation to verify
10. Create supernet using calculateSubnet()

**Helper functions to implement:**
- `isPowerOfTwo(n: number): boolean` - Same as subnetting.ts (can duplicate or extract to shared module)
- `addressToNumber(address: string, ipVersion: 4 | 6): bigint` - Convert IP to numeric for comparison
- `compareAddresses(a: string, b: string, ipVersion: 4 | 6): number` - Compare two addresses
- `getBlockSize(ipVersion: 4 | 6, cidr: number): bigint` - Calculate address block size
- `addToAddress(address: string, offset: bigint): string` - Add offset to IP address

**Error handling (return success: false with error message):**
- "Need at least 2 networks to aggregate"
- "Number of networks must be a power of 2"
- "Invalid CIDR notation: {network}"
- "All networks must be same IP version"
- "All networks must have same CIDR prefix"
- "First network must be on /{newCidr} boundary"
- "Networks not contiguous: expected {expected}, got {actual}"

Use ipaddr.js for IP address parsing. Import calculateSubnet from ./subnet-calculator, SubnetResult from ./types.
