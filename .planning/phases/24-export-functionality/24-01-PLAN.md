  <name>Task 1: Create CSV export utility</name>
  <files>src/lib/utils/csv-export.ts</files>
  <action>
Create `src/lib/utils/csv-export.ts` with the following features:

1. **Interfaces:**
   - `CsvExportOptions`: filename (optional), includeHeaders (default true)
   - `CsvRow`: Record with string keys and string/number/boolean/null values

2. **Main function `exportToCsv(data: CsvRow[], options?: CsvExportOptions)`:**
   - Return early if data is empty (with console.warn)
   - Extract headers from first row's keys
   - Build CSV content with proper escaping
   - Add UTF-8 BOM (`\uFEFF`) for Excel compatibility
   - Create Blob with type `text/csv;charset=utf-8;`
   - Trigger download via hidden anchor element
   - Clean up with `URL.revokeObjectURL()` to prevent memory leaks

3. **Helper function `escapeCSV(value, delimiter)`:**
   - Handle null/undefined as empty string
   - Sanitize CSV injection (prefix `=`, `+`, `-`, `@` with single quote)
   - Wrap in quotes if contains delimiter, quotes, or newlines
   - Double any internal quotes

4. **Helper function `downloadBlob(blob, filename)`:**
   - Create object URL
   - Create hidden anchor element
   - Trigger click
   - Cleanup with setTimeout (100ms) for cross-browser compatibility

Follow the pattern from 24-RESEARCH.md code examples.
