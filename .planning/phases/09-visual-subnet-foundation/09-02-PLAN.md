    **Function 1: parseIPInput(input: string, subnetMask?: string): ParsedInput**
    - Try CIDR notation first (e.g., "192.168.1.0/24", "2001:db8::/32")
    - If slash detected, validate with ipaddr.isValidCIDR()
    - Split on slash, parse CIDR number, validate range (0-32 for IPv4, 0-128 for IPv6)
    - If no slash and subnetMask provided, convert mask to CIDR using ipaddr.IPv4.parse().prefixLengthFromSubnetMask()
    - Throw descriptive errors for invalid inputs (function can throw, store will catch)
    - Return { ipAddress, cidr, format: 'cidr' | 'mask' }

    **Function 2: validateIPAddress(ip: string): IPValidationResult**
    - Use ipaddr.isValid(ip) for validation
    - If valid, parse with ipaddr.parse(ip) and check .kind() for version
    - Return { valid: true, version: 4 | 6 } or { valid: false, version: null, error: string }

    **Error messages:**
    - "Invalid CIDR notation" for malformed CIDR
    - "CIDR must be between 0 and 32 for IPv4" (or 0-128 for IPv6)
    - "Invalid IP address format"
    - "Subnet mask notation only supported for IPv4" (if IPv6 with mask)
    - "Invalid subnet mask" for malformed subnet masks

    Follow project error handling pattern: throw errors from pure functions, let caller handle.

<task type="auto">
  <name>Implement subnet calculation functions</name>
  <files>src/lib/converters/network/subnet-calculator.ts</files>
  <action>
    Create subnet calculation logic following RESEARCH.md Pattern 1. Handle IPv4/IPv6 differences.

    **Import dependencies:**
    ```typescript
    import ipaddr from 'ipaddr.js';
    import type { SubnetResult } from './types';
    import { parseIPInput } from './ip-parser';
    ```

    **Main function: calculateSubnet(ipAddress: string, cidr: number): SubnetResult**
    - Parse IP address with ipaddr.parse()
    - Check addr.kind() to determine IPv4 vs IPv6
    - Delegate to calculateIPv4Subnet() or calculateIPv6Subnet()

    **IPv4 calculation (private function):**
    - Network address: ipaddr.IPv4.networkAddressFromCIDR(`${addr}/${cidr}`)
    - Broadcast address: ipaddr.IPv4.broadcastAddressFromCIDR(`${addr}/${cidr}`)
    - Subnet mask: ipaddr.IPv4.subnetMaskFromPrefixLength(cidr)
    - Host count: totalHosts = BigInt(2) ** BigInt(32 - cidr)

    **Special cases for IPv4:**
    - /32 (single host): usableHosts = 1, firstUsable = lastUsable = address
    - /31 (point-to-point, RFC 3021): usableHosts = 2, no network/broadcast reservation
    - Normal: usableHosts = totalHosts - 2, firstUsable = network + 1, lastUsable = broadcast - 1

    **IPv6 calculation (private function):**
    - Network address: calculate by zeroing host bits (bitwise operations on addr.parts array)
    - Last address: calculate by setting all host bits to 1
    - No broadcast address (return null)
    - No subnet mask (return null)
    - Host count: totalHosts = BigInt(2) ** BigInt(128 - cidr)
    - usableHosts = totalHosts (no network/broadcast reservation, except /128 = 1)

    **IPv6 bitwise helpers (private functions):**
    - calculateIPv6NetworkAddress(parts: number[], cidr: number): number[]
      - Zero out parts after network portion
      - Handle partial parts with bit masking
    - calculateIPv6LastAddress(parts: number[], cidr: number): number[]
      - Set all parts after network portion to 0xFFFF
      - Handle partial parts with bit masking

    Follow RESEARCH.md code examples exactly for IPv6 bitwise operations.

    **Return SubnetResult with all fields populated correctly based on IP version.**

    IMPORTANT pitfalls to avoid (from RESEARCH.md):
    - Don't apply 2^n - 2 formula to /31 and /32 subnets
    - Don't calculate broadcast for IPv6 (return null)
    - Use BigInt for all host count calculations
    - IPv6 firstUsable === networkAddress (no reservation)
  </action>
