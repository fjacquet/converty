- [ ] URL sync consolidated: Middleware used (not manual URLSearchParams)

- [ ] No hard-coded strings (should use translations)
- [ ] Type definitions shared (exported from lib/converters)

**Functional Programming Review (QUAL-07):**

Focus areas:

- src/lib/converters/ (MUST be pure functions)
- State updates in stores (MUST be immutable)
- No shared mutable state (MUST be isolated)

Checklist:

- [ ] Calculation functions pure: No side effects, deterministic
- [ ] No I/O in calculations: No console.log, fetch, localStorage
- [ ] No mutations: Input parameters not modified
- [ ] Zustand state updates immutable: Uses spread operators
- [ ] No global mutable variables

Commands to help:

