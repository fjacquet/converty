  Create `src/lib/converters/engineering/stress-strain.ts` following the pure function pattern:

  **Input interface:**
  ```typescript
  export interface StressStrainInput {
    /** Calculation mode */
    mode: "stress" | "strain" | "youngs-modulus";
    /** Applied force in kN */
    force: number;
    /** Cross-sectional area in mm2 */
    area: number;
    /** Original length in mm */
    originalLength: number;
    /** Change in length in mm */
    changeInLength: number;
    /** Material ID for property lookup (optional) */
    materialId: string;
    /** Custom Young's modulus in GPa (used if no material selected) */
    customYoungsModulus: number;
    /** Custom yield strength in MPa (used if no material selected) */
    customYieldStrength: number;
  }
  ```

  **Result interface:**
  ```typescript
  export interface StressStrainResult {
    /** Engineering stress in MPa */
    stress: number;
    /** Engineering strain (dimensionless) */
    strain: number;
    /** Young's modulus in GPa */
    youngsModulus: number;
    /** Safety factor (yield strength / stress), null if no yield strength available */
    safetyFactor: number | null;
    /** Whether stress exceeds yield strength */
    exceedsYield: boolean;
    /** Material name (if selected) */
    materialName: string | null;
    /** Yield strength used in MPa */
    yieldStrength: number | null;
    /** Step-by-step calculation breakdown */
    steps: string[];
    /** Stress in additional units for display */
    stressUnits: {
      mpa: number;
      gpa: number;
      psi: number;
      ksi: number;
    };
  }
  ```

  **Calculate function logic:**
  1. Load material from materials.json if materialId provided (use dynamic import or pass materials data)
  2. Based on mode:
     - "stress": Calculate sigma = F/A (force in N = kN * 1000, area in mm2, result in MPa)
       - Then derive strain = sigma / (E * 1000) if E available, and E from material or custom
     - "strain": Calculate epsilon = deltaL / L
       - Then derive stress = epsilon * E * 1000 if E available
     - "youngs-modulus": Calculate E = sigma / epsilon (result in GPa)
       - Need both force/area (for stress) and lengths (for strain)
  3. Calculate safety factor = yieldStrength / stress (if yield strength available and stress > 0)
  4. Determine exceedsYield = stress > yieldStrength
  5. Convert stress to multiple units:
     - MPa (base)
     - GPa = MPa / 1000
     - psi = MPa * 145.038
     - ksi = psi / 1000
  6. Build steps array showing each formula and intermediate value

  **Validation:**
  - Return null if area <= 0 or originalLength <= 0
  - Return null if force is negative
  - Handle division by zero for strain calculation

  **Important conversion notes:**
  - Force input in kN, convert to N: N = kN * 1000
  - Stress = Force(N) / Area(mm2) = result in MPa (since 1 MPa = 1 N/mm2)
  - Young's modulus stored in GPa. For stress calculation: stress = strain * E * 1000 (GPa to MPa)

  Also provide a helper function to load materials:
  ```typescript
  import materialsData from "@/data/engineering/materials.json";
  import type { Material } from "./types";

  export function getMaterials(): Material[] {
    return materialsData as Material[];
  }

  export function getMaterialById(id: string): Material | undefined {
    return getMaterials().find(m => m.id === id);
  }
  ```
