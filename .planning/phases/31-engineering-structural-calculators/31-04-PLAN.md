  **Formulas by beam type and load type:**

  All formulas below use: P = point load (N), w = distributed load (N/mm), L = length (mm), E = Young's modulus (MPa = N/mm2), I = moment of inertia (mm4), a = load position from left (mm), b = L - a.

  IMPORTANT: Convert input units before calculation:
  - P: kN to N (* 1000)
  - w: kN/m to N/mm (* 1000 / 1000 = * 1)... actually kN/m to N/mm = kN/m / 1000, no wait:
    kN/m = 1000 N / 1000 mm = 1 N/mm. So w_kN_per_m = w_N_per_mm.
  - E: GPa to MPa (* 1000), and MPa = N/mm2
  - Results: moments in N.mm, convert to kN.m at output (/ 1e6)
  - Stress: sigma = M * c / I where c = depth/2. Result in MPa (N/mm2).

  **1. Simply Supported Beam + Point Load at position a:**
  - Reactions: R_A = P*b/L, R_B = P*a/L
  - Max moment: M_max = P*a*b/L (at load position)
  - Max deflection (centered load, a=L/2): delta = P*L^3 / (48*E*I)
  - Max deflection (off-center): delta = P*a^2*b^2 / (3*E*I*L) at position x = sqrt(a*(a+2b)/3) from A
  - Shear: V(x) = R_A for x < a, V(x) = R_A - P for x > a
  - Moment: M(x) = R_A*x for x <= a, M(x) = R_A*x - P*(x-a) for x > a

  **2. Simply Supported Beam + UDL:**
  - Reactions: R_A = R_B = w*L/2
  - Max moment: M_max = w*L^2/8 (at midspan)
  - Max deflection: delta = 5*w*L^4 / (384*E*I) (at midspan)
  - Shear: V(x) = w*L/2 - w*x
  - Moment: M(x) = w*L*x/2 - w*x^2/2

  **3. Cantilever + Point Load at free end:**
  - Reaction: R_A = P (at fixed end)
  - Fixed-end moment: M_A = P*L (at fixed end)
  - Max moment: M_max = P*L (at fixed end)
  - Max deflection: delta = P*L^3 / (3*E*I) (at free end)
  - Shear: V(x) = -P (constant along beam, sign convention: fixed end at left)
  - Moment: M(x) = -P*(L-x) (linear, max at fixed end x=0)

  **4. Cantilever + UDL:**
  - Reaction: R_A = w*L
  - Fixed-end moment: M_A = w*L^2/2
  - Max moment: M_max = w*L^2/2 (at fixed end)
  - Max deflection: delta = w*L^4 / (8*E*I) (at free end)
  - Shear: V(x) = -w*(L-x) (at fixed end at left)
  - Moment: M(x) = -w*(L-x)^2/2

  **5. Fixed-Fixed + Point Load at center (a=L/2):**
  - Reactions: R_A = R_B = P/2
  - Fixed-end moments: M_A = M_B = P*L/8
  - Max moment: P*L/8 (at supports and at load point, moment at midspan = P*L/8 positive)
  - Max deflection: delta = P*L^3 / (192*E*I) (at midspan)
  - For off-center: use simplified centered formula (acceptable for MVP)

  **6. Fixed-Fixed + UDL:**
  - Reactions: R_A = R_B = w*L/2
  - Fixed-end moments: M_A = M_B = w*L^2/12
  - Max positive moment: w*L^2/24 (at midspan)
  - Max deflection: delta = w*L^4 / (384*E*I) (at midspan)
  - Shear: V(x) = w*L/2 - w*x
  - Moment: M(x) = w*L^2/12 - w*L*x/2 + w*x^2/2 (note: M at supports = -wL^2/12)
    Actually more precisely: M(x) = -w*L^2/12 + w*L*x/2 - w*x^2/2

  **Diagram generation:**
  Generate 50 evenly spaced points along the beam (x from 0 to 1 normalized). For each point, calculate V(x) and M(x) using the appropriate formulas above. Also calculate deflection along the beam using standard beam deflection equations (integration of M/(EI)).

  For deflection along simply-supported + point load: Use standard deflection equation for two segments.
  For simply-supported + UDL: delta(x) = w*x/(24*E*I) * (L^3 - 2*L*x^2 + x^3)
  For cantilever + point load: delta(x) = P*x^2/(6*E*I) * (3*L - x)
  For cantilever + UDL: delta(x) = w*x^2/(24*E*I) * (x^2 - 4*L*x + 6*L^2)
  For fixed-fixed + UDL: delta(x) = w*x^2/(24*E*I) * (L - x)^2

  **Stress calculation:**
  sigma = M_max * c / I where c = sectionDepth/2 (distance from neutral axis to extreme fiber)
  Result in MPa (N/mm2). If sectionDepth is 0, skip stress calculation.

  **Safety factor:**
  If yieldStrength available and stress > 0: SF = yieldStrength / sigma
  exceedsYield = sigma > yieldStrength

  **Material and section lookup:**
  Reuse getMaterialById() and getBeamSectionById() from stress-strain.ts and moment-of-inertia.ts.
  Import them or duplicate the simple helper functions.
  When material selected: use its youngsModulus
  When section selected: use its Ix as moment of inertia, its depth as sectionDepth

  **Validation:**
  - Return null if beamLength <= 0
  - Return null if E <= 0 or I <= 0
  - Return null if both pointLoad and distributedLoad are 0
  - For point load: loadPosition must be 0 < loadPosition < beamLength (for simply-supported/fixed-fixed)
  - For cantilever: load at free end (loadPosition = beamLength, or ignored)

  Update `src/lib/converters/engineering/index.ts` to export beam-deflection functions.
