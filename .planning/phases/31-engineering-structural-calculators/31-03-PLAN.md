  **Calculate function - shape formulas:**

  Each shape has well-known formulas. Implement each precisely:

  - **Rectangle:** Ix = bh^3/12, Iy = hb^3/12, A = bh, Zx = bh^2/6, Zy = hb^2/6
  - **Circle:** Ix = Iy = pi*d^4/64, A = pi*d^2/4, Zx = Zy = pi*d^3/32
  - **Hollow rectangle:** Ix = (BH^3 - bh^3)/12 where b=B-2t, h=H-2t. A = BH - bh
  - **Hollow circle:** Ix = Iy = pi*(D^4 - d^4)/64. A = pi*(D^2 - d^2)/4
  - **I-beam:** Use three rectangles approach:
    - Top flange: bf x tf centered at top
    - Web: tw x (h - 2*tf) centered
    - Bottom flange: bf x tf centered at bottom
    - Use parallel axis theorem: I_total = sum(I_ci + A_i * d_i^2)
    - Centroid is at mid-height (symmetric)
  - **T-beam:** Two rectangles (flange + web), NOT symmetric:
    - Find centroid first: y_bar = sum(A_i * y_i) / sum(A_i)
    - Then apply parallel axis theorem for Ix
  - **Channel (C-shape):** Three rectangles (two flanges + web):
    - Symmetric about x-axis (centroid at mid-height)
    - NOT symmetric about y-axis (centroid offset toward web)
  - **Angle (L-shape):** Two rectangles (vertical leg + horizontal leg):
    - NOT symmetric about either axis
    - Find centroid for both axes, then parallel axis theorem

  For "standard" mode: Load beam-sections.json, find section by ID, return pre-computed values directly. Still populate dimensions for SVG rendering based on section type.

  Add helper:
  ```typescript
  import sectionsData from "@/data/engineering/beam-sections.json";
  import type { BeamSection } from "./types";

  export function getBeamSections(): BeamSection[] {
    return sectionsData as BeamSection[];
  }

  export function getBeamSectionById(id: string): BeamSection | undefined {
    return getBeamSections().find(s => s.id === id);
  }
  ```
