    Binary conversion helper function (in same file):
    ```typescript
    function convertToBinary(addr: ipaddr.IPv4 | ipaddr.IPv6, ipVersion: 4 | 6): string[] {
      if (ipVersion === 4) {
        const octets = (addr as ipaddr.IPv4).octets;
        return octets.flatMap(octet =>
          octet.toString(2).padStart(8, "0").split("")
        );
      } else {
        const parts = (addr as ipaddr.IPv6).parts;
        return parts.flatMap(part =>
          part.toString(2).padStart(16, "0").split("")
        );
      }
    }
    ```

    Render structure:
    1. IP Address Binary section:
       - Parse IP address using ipaddr.parse(ipAddress)
       - Convert to binary array
       - Map through bits with conditional styling:
         - Network bits (index < result.cidr): blue background (bg-blue-100 text-blue-700 dark:bg-blue-900/30)
         - Host bits (index >= result.cidr): green background (bg-green-100 text-green-700 dark:bg-green-900/30)
       - Each bit in span with px-1 py-0.5 rounded
       - Use role="list" on container, role="listitem" on each span
       - Add aria-label to each bit for accessibility

    2. Subnet Mask Binary section (IPv4 only):
       - Render only if result.ipVersion === 4 && result.subnetMask
       - Generate mask bits: Array.from({ length: totalBits }, (_, i) => i < result.cidr ? "1" : "0")
       - Style: "1" bits blue, "0" bits gray
       - Same ARIA structure as IP address section

    3. Bit position markers:
       - IPv4: Show 0-7, 8-15, 16-23, 24-31 (octet boundaries)
       - IPv6: Show 0-15, 16-31, ..., 112-127 (16-bit group boundaries)
       - Use flex justify-between with text-xs text-muted-foreground

    Responsive design:

    - Add gap-1 between bits
